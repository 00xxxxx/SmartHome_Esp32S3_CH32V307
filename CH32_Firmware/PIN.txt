# -------------------------------------------------------------------
# Core Modules
# -------------------------------------------------------------------
## 串口 UART1 (用于调试信息输出 和 接收远程指令)
Tx1: PA9  -> 连接到串口调试工具的RX端
Rx1: PA10 <- 连接到串口调试工具的TX端

## 串口 UART2 (连接CC2530协调器)
Tx2: PA2
Rx2: PA3

## 以太网 (Ethernet - RMII Interface)
- RJ45接口 <-> CH32V307 (RMII模式引脚，由开发板硬件决定，无需软件配置)

## 调试接口 (Debug)
- SWDIO/TMS: PA13
- SWCLK/TCK: PA14

# -------------------------------------------------------------------
# Sensors (传感器模块)
# -------------------------------------------------------------------
## DHT11 (温湿度传感器)
- DATA: PC1 (GPIO输入/输出，单总线协议)

## 光敏电阻 (Photoresistor)
- AOUT: PA1 (ADC1/2 通道1，模拟量输入，用于检测环境亮度)

# -------------------------------------------------------------------
# Actuators & User Interface (执行器与用户交互)
# -------------------------------------------------------------------
## LED (指示灯)
- LED1: PC3 (GPIO推挽输出，低电平点亮，用于光感/手动控制)
- LED2: PC4 (GPIO推挽输出，低电平点亮，用于串口远程控制)

## KEY (按键)
- KEY: PA0 (GPIO上拉输入，连接到EXTI0，下降沿触发中断)

## 蜂鸣器 (Buzzer)
- BEEP: PA5 (GPIO推挽输出，高电平鸣叫，用于报警)

## 门锁 (舵机 - Servo SG90)
- PWM:  PC5 (TIM3_CH2 PWM输出，用于控制舵机角度)

# ===================================================================
# CC2530 终端节点 (传感器)
# ===================================================================
## SR602 (人体红外感应模块)
- DOUT: P0.0 (GPIO输入，高电平表示检测到人体)

## MQ2 (烟雾/可燃气体传感器)
- AOUT: P0.1 (模拟输入，用于检测烟雾浓度)
